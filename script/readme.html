INSTALLATION INSTRUCTIONS

1
unzip the ZIP file in a folder on your computer.

2
upload all files in a folder on your website. Please note that you can upload the files in any folder but we will use "script" folder name in the instructions below.

IMPORTANT: you may need to upload the files in binary mode. Also depending on your server configuration you may need to set write permissions to options file and few of the script folders. The install wizard will advise you the exact files and folders which need to be set. You can watch video tutorials how to upload in binary mode and set write permissions using some of the most popular FTP clients here.

3
open http://www.REPLACEYOURWEBSITE.com/script/, where www.REPLACEYOURWEBSITE.com is the domain name for your website. Installation wizard will ask you to complete few steps and will install the script
